# Você Seguradora — landing page

Landing page da Você Seguradora, construída sobre o manual de identidade visual oficial da marca.

**Demo:** https://lael83.github.io/voce-seguradora/

## Seções

- **Hero fullscreen** — fotografia de ciclismo com pan contínuo e faixas de velocidade neon
- **Letreiro cinético** — "mais saúde · mais movimento · mais proteção"
- **Movimento** — frases oficiais do manual sobre fotografia de esporte
- **Simulador Residencial** — CEP, importância segurada e planos Essence/Gold/Prime com estimativa na hora
- **Catálogo** — bento grid dos 13 produtos (PF/PJ)
- **Transparência** — o que está e o que não está coberto, lado a lado
- **Públicos** — cliente, corretor, parceiro
- **Área do corretor**, tagline pincelada, aviso antifraude e rodapé SUSEP

## Rodando

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # gera dist/
```

## Stack

React 19 · TypeScript · Vite 6 · Tailwind
